# Trader Performance vs Market Sentiment Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear/Greed) affects trader behavior and profitability.

## Datasets
1. Bitcoin Fear & Greed Index
2. Hyperliquid Historical Trader Data

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Analysis Performed
- Data cleaning
- Sentiment vs trader performance analysis
- Trade frequency analysis
- Profit analysis
- Long vs Short behavior analysis

## Key Insights
- Trader profitability varies depending on market sentiment.
- Trading activity increases during Fear and Extreme Greed periods.
- Traders tend to take larger positions during bullish sentiment.

## Strategy Recommendations
1. Reduce trade size during Fear periods to manage risk.
2. Increase trading activity cautiously during Greed periods.
